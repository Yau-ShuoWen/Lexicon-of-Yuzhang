## 结构

**没有纯正的专用数据传输层**

### 数据访问层（DAO）

| 类名             | 描述             | 说明                      |
|----------------|----------------|-------------------------|
| `HanziEntity`  | 字表的主表的model    | 特殊字段只读                  |
| `HanziPinyin`  | 字表的读音变体表的model | ╱ 查询的时候的特殊字段直接由SQL语句合成  |
| `HanziSimilar` | 字表的相似汉字表的model | ╲ 不是由这两个DAO聚合而来，这是为了编辑的 |

### 领域模型层（Domain）

| 类名            | 描述         | 说明                         |
|---------------|------------|----------------------------|
| `HanziItem`   | 查询汉字字的主要载体 |                            |
| `HanziGroup`  | 汉字的组，聚合结果  | 汉字的特殊性，不能直接使用数组存           |
| `HanziShow`   | 字的详细信息，用于读 | 兼做DTO 后端→前端 查询数据           |
| `HanziUpdate` | 更新资料的载体    | 兼做DTO 后端→前端 原数据  前端→后端 新数据 |
| `HanziCreate` | 批量初始化的载体   | 兼做DTO 前端→后端 新数据            |

## 流程

| 流程     | 描述                                                                 | 说明 |
|--------|--------------------------------------------------------------------|----|
| 搜索-筛选  | `HanziEntity`→`HanziItem`→`HanziGroup`→`SearchResult`→前端           |    |
| 搜索-特定  | `HanziEntity`→`HanziItem`→`HanziGroup`→`HanziShow`→前端              |    |
| 编辑-筛选  | `HanziEntity`→`HanziOutline`→前端                                    |    |
| 编辑-原数据 | `HanziEntity` + `HanziPinyin` + `HanziSimilar` → `HanziUpdate` →前端 |    |
| 编辑-新数据 | 前端→ `HanziUpdate` → `HanziEntity` + `HanziPinyin` + `HanziSimilar` |    |
| 初始化    | 前端→ `HanziCreate` → `HanziEntity` + `HanziPinyin`                  |    |
