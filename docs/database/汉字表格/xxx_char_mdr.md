# 概述

普通话和方言对照表，`xxx`为当地方言代号，详见[这篇文档](../../命名/方言代号命名.md)

# 结构

| 列名            | 内容          | 数据说明              |
|---------------|-------------|-------------------|
| `mandarin_id` | 🔍 外键-普通话字表 | 连接的是`mdr_char.id` |
| `dialect_id`  | 🔍 外键-方言字表  | 连接的是`xxx_char.id` |

# 数据限定

- 普通话字表最多出现一次，方言字表可以出现很多次
- 这种设计使得在标注读音的时候，一个普通话读音可以完全对应方言的词条

例如

- 普通话多音字多对一读音

| 普通话字表   | 方言字表    |
|---------|---------|
| 假 `jiǎ` | 假 `ga3` | 
| 假 `jià` | 假 `ga3` | 

- 普通话多个汉字对应一个词条

| 普通话字表   | 方言字表      |
|---------|-----------|
| 车 `chē` | 车-車 `ca1` | 
| 車 `chē` | 车-車 `ca1` |
