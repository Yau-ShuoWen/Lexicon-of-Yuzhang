| id      | 🔍 自增主键    | 自动生成列    |

| sc      | 📖 标签-简体中文 |          |
| tc      | 📖 标签-繁体中文 |          |
| pinyin  | 📢 读音      |          |
| sort    | 📊 排列顺序    | 数字从1开始   |
