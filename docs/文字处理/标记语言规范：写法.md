# 拼音类内容：使用`[]`

## 说明

1. 使用半角方括号
2. 文本内部若有空格，会分开解析，如`[lan4 cong1]`等于`[lan4] [cong1]`，`[+lan4 cong1]`等于`[+lan4] [+cong1]`。除此之外单个拼音内部不能空格

## 写法

### 方言拼音

- 标识：直接使用中括号包裹即可
- 拼音：使用方言拼音的输入版本。

> - 编辑者写的：`[gon1]`
> - 用户看到的：`gòn`

### 国际音标（音系内部）

- 标识：在左括号后加一个`*`
- 拼音：使用方言拼音的输入版本。

> - 编辑者写的：`[*gon1]`
> - 偏好为「方言拼音」用户看到的：`gòn`
> - 偏好为「国际音标」用户看到的：`kɔŋ˦˨`

### 国际音标（音系外部）

- 标识：在左括号后加一个`-`
- 拼音：左侧写汉语学界的国际音标，右侧写音调的五度标记法，中间用`-`

> - 编辑者写的：`[*ɕyia_42]`
> - 用户看到的：`ɕyia˦˨`

### 汉语拼音

- 标识：在左括号后加一个`+`
- 拼音：使用汉语拼音输入版本。

> - 编辑者写的：`[+yi1]`
> - 用户看到的：`yī`

### 自定义拼音

- 标识：在左括号后加一个`#`
- 拼音：左侧写上述音系外部的国际音标，右侧写造出来的方言拼音，中间用`=`

> - 编辑者写的：`[#sio_45=siō]`
> - 偏好为「方言拼音」用户看到的：`siō`
> - 偏好为「国际音标」用户看到的：`sio˧˥`

### 其他拼音

- 标识：在左括号后加一个`/`
- 拼音：里面可以有任何拉丁系符号，仅为了改变字体强调这是其他形式的拼音，所以不会检查是否合理

> - 编辑者写的：`[/aoeiuü]`
> - 用户看到的：`aoeiuü`

# 注释：使用`/**/`

1. 只会在编辑文本框中看见，不作为最终显示
2. 可以用于结构校对随笔

> - 编辑者写的：`所以结论为此。/*此处存疑，需要更多信息*/`
> - 用户看到的：`所以结论为此。`

| 格式名稱 | x是什麼字母     | 完成 |
|------|------------|----|
| 標題   | h(header)  |    |

| 註解   | n(note)    | ✓  |

| 內部鏈接 | l(link)    |    |
| 外部鏈接 | u(url)     |    |
| 外部鏈接 | t(table)   |    |
| 音頻   | a(audio)   |    |

## 内容

### 粗体

> - 编辑者写的：`这里用法{b 不同于}普通话`
> - 用户看到的：这里用法**不同于**普通话

### 斜体

> - 编辑者写的：`我也不知道什么时候会用上{b 斜体}的写法`
> - 用户看到的：我也不知道什么时候会用上*斜体*的写法

### 注解

> 主要用在辞书里，因为说明的可以直接跟在后面。
> - 编辑者写的：`他话{n 说}什哩`
> - 用户看到的：他话^什哩 鼠标在^上 原书注：说

## 特殊格式的内容：使用`{}`

| 名称             | 结构范例              | 显示效果                                                                                                                | 说明               |
|----------------|-------------------|---------------------------------------------------------------------------------------------------------------------|------------------|
| 关键词            | 使用大括号`{}`         | 编辑者写的：`{你}：代詞，用于…… `<br/>  用户看到的：<br/>词条「你」下的附注：《南昌方言词典》 你 代词，用于……                                                  | 用于自动绑定           |
| 关键词<br/>（多个）   | 使用大括号`{}`         | 编辑者写的：`{医生}：又称{郎中}`<br/> <br/> 用户看到的：<br/> 词条「医生」下的附注：《南昌方言词典》 医生：又称郎中<br/>词条「郎中」下的附注：《南昌方言词典》 医生：又称郎中              | 直接可以多个绑定         |
| 关键词<br/>（隐藏绑定） | 关键词前面加横杠          | 编辑者写的：`{中秋}：……{-中秋节}`<br/>用户看到的：<br/> 词条「中秋」下的附注：《南昌方言词典》 中秋：……<br/>词条「中秋节」下的附注：《南昌方言词典》 中秋：……                      | 关键词不一定会出现在内容里    |

| 粗体             | 用`b`标识（bold）      | 编辑者写的： `这里用法{b 不同于}普通话` <br/> 用户看到的：                                                                                |                  |
