# 拼音类内容：使用`[]`

## 说明

1. 使用半角方括号
2. 文本内部允许有空格，会单独解析，如`[lan4 cong1]`等于`[lan4] [cong1]`，`[+lan4 cong1]`等于`[+lan4] [+cong1]`，拼音内部不能空格

## 写法

### 方言拼音

- 标识：直接使用中括号包裹即可
- 拼音：使用方言拼音的输入版本。

> - 编辑者写的：`[gon1]`
> - 用户看到的：`gòn`

### 国际音标（音系内部）

- 标识：在左括号后加一个`*`
- 拼音：使用方言拼音的输入版本。

> - 编辑者写的：`[*gon1]`
> - 偏好为「方言拼音」用户看到的：`gòn`
> - 偏好为「国际音标」用户看到的：`kɔŋ˦˨`

### 国际音标（音系外部）

- 标识：在左括号后加一个`-`
- 拼音：左侧写汉语学界的国际音标，右侧写音调的五度标记法，中间用`-`

> - 编辑者写的：`[*ɕyia_42]`
> - 用户看到的：`ɕyia˦˨`

### 汉语拼音

- 标识：在左括号后加一个`+`
- 拼音：使用汉语拼音输入版本。

> - 编辑者写的：`[+yi1]`
> - 用户看到的：`yī`

### 自定义拼音

- 标识：在左括号后加一个`#`
- 拼音：左侧写上述音系外部的国际音标，右侧写造出来的方言拼音，中间用`=`

> - 编辑者写的：`[#sio_45=siō]`
> - 偏好为「方言拼音」用户看到的：`siō`
> - 偏好为「国际音标」用户看到的：`sio˧˥`

### 其他拼音

- 标识：在左括号后加一个`/`
- 拼音：里面可以有任何拉丁系符号，仅为了改变字体强调这是其他形式的拼音，所以不会检查是否合理

> - 编辑者写的：`[/aoeiuü]`
> - 用户看到的：`aoeiuü`

<br><br><br><br>

# 注释：使用`/**/`

1. 只会在编辑文本框中看见，不作为最终显示
2. 可以用于结构校对随笔

> - 编辑者写的：`所以结论为此。/*此处存疑，需要更多信息*/`
> - 用户看到的：`所以结论为此。`

<br><br><br><br>

# 富文本：使用`{x }`

| 格式名称 | x是什么字母     | 内层嵌套 | 外层嵌套 | 说明                  | 代码完成 |
|------|------------|------|------|---------------------|------|
| 标题   | h(header)  | ×    | ×    | 标题是单独的              | ✓    |
| 粗体   | b(bold)    | ×    | ✓    | 字体格式，衹能作为最里一层       | ✓    |
| 斜体   | i(italic)  | ×    | ✓    | 字体格式，衹能作为最里一层       | ✓    |
| 删除划线 | d(delete)  | ×    | ✓    | 字体格式，衹能作为最里一层       | ✓    |
| 词语链接 | c(ciyu)    | ✓    | ✓    | 链接的字体部分可以有格式        | ✓    |
| 链接   | l(link)    | ✓    | ✓    | 链接的字体部分可以有格式        | ✓    |
| 注解   | n(note)    | ✓    | ×    | 注解必须在最外层            | ✓    |
| 保护文本 | s(save)    | -    | -    | 后端不受简繁转换等的影响，不出现在前端 |      |
| 音频   | a(audio)   | ×    | ×    | 设计中                 |      |
| 图片   | p(picture) | ×    | ×    | 设计中                 |      |
| 视频   | v(video)   | ×    | ×    | 设计中                 |      |

## 内容

### 标题

说明略

### 粗体

> - 编辑者写的：`这里用法{b 不同于}普通话`
> - 用户看到的：这里用法**不同于**普通话

### 斜体

> - 编辑者写的：`我也不知道什么时候会用上{b 斜体}的写法`
> - 用户看到的：我也不知道什么时候会用上*斜体*的写法

### 删除划线

> - 编辑者写的：`这是一个姚说文{d 神经}的想法`
> - 用户看到的：`这是一个姚说文~~神经~~的想法

### 词语链接

- 可以导向对应词汇
> - 编辑者写的：`{c 南昌}`
> - 用户看到的：[南昌](https://yuzhangci.com/sc/lac/c/南昌) ---> 创建sc/lac/c/南昌 的链接
- 可以重定向
> - 编辑者写的：`{c 豫章|南昌}`
> - 用户看到的：[豫章](https://yuzhangci.com/sc/lac/c/南昌) ---> 还是创建sc/lac/c/南昌 的链接

### 链接

> - 编辑者写的：`{l 这是一个链接|https://yuzhangci.com/sc/lac/ysw}`
> - 用户看到的：[这是一个链接](https://yuzhangci.com/sc/lac/ysw)

### 注解

> 主要用在辞书里，因为说明的可以直接跟在后面。
> - 编辑者写的：`他话{n 说}什哩`
> - 用户看到的：他话^什哩 鼠标在^上 原书注：说

# 跨行：行首标注

### 无序列表

- 标识：行开头`-`，需要空格

```
- 第一条
- 第二条
- 第三条
```

效果：

- 第一条
- 第二条
- 第三条

### 有序列表

- 标识：行开头`+`，需要空格

```
+ 第一条
+ 第二条
+ 第三条
```

效果：

1. 第一条
2. 第二条
3. 第三条

### 句式列表

- 标识：行开头`*`，需要空格

```
* 他喜欢这个
* 渠作兴箇只
* [jie2 zok6 xin0 go3 za0]
```

效果：

- 例句：他喜欢这个
- 说法：渠作兴箇只
- 拼音：jié zők xin gǒ za
