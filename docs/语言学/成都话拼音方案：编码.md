# 号码

六位长度
```
声母*2 韵尾 介母 中心元音 儿化
```

- ### 前两位：声母

| 编码   | 对应音节 |
|:-----|:-----|
| `00` | 零声母  |
| `01` | `b`  |
| `02` | `p`  |
| `03` | `m`  |
| `04` | `f`  |
| `05` | `v`  |
| `06` | `d`  |
| `07` | `t`  |
| `08` | `n`  |
| `09` | `g`  |
| `10` | `k`  |
| `11` | `ng` |
| `12` | `h`  |
| `13` | `j`  |
| `14` | `q`  |
| `15` | `nh` |
| `16` | `x`  |
| `17` | `z`  |
| `18` | `c`  |
| `19` | `s`  |
| `20` | `r`  |


| 编码  | 对应音节      |
|:----|:----------|
| `0` | 开元音韵尾/无韵尾 |
| `1` | z c s的韵母  |
| `2` | 韵母er      |
| `3` | 韵母 m      |
| `4` | i 尾       |
| `5` | u 尾       |
| `6` | n 尾       |
| `7` | ng 尾      |
| `8` | r 尾       |

- ### 第四位：介韵母

| 编码  | 对应音节     |
|:----|:---------|
| `0` | 开口呼      |
| `1` | 合口呼：`i`  |
| `2` | 闭口呼：`u`  |
| `3` | 撮口呼：`yu` |
