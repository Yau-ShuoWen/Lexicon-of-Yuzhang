## 拼音类内容：使用`[]`

| 名称              | 结构范例             | 显示效果                                      | 说明                                 |
|-----------------|------------------|-------------------------------------------|------------------------------------|
| 方言拼音            | 使用中括号`[]`        | 编辑者写的：`[gon1]` <br/>  用户看到的：`gòn`         | 使用打字版本，系统自动变更写法                    |
| 国际音标<br/>（音系内部） | 音节前使用星号          | 编辑者写的：`[*gon1]` <br/>  用户看到的：`kɔŋ˦˨`      | 直接把他**当做方言拼音**拼写，系统自动变更写法为对应音标     |
| 国际音标<br/>（音系外部） | 音节和音调之间用-连接      | 编辑者写的：`[kɔŋ-42]` <br/>  用户看到的：`kɔŋ˦˨`     | 一般不使用，只有超出音系的音标才使用<br/>五度标记法使用数字代替 |
| 汉语拼音            | 音节前使用斜杠          | 编辑者写的：`[/yī]`<br/> 用户看到的：`yī`             | 系统不会修改格式，只会修改字体，<br/>请使用标准的上标写法    |
| 其他拼音            | 音节前使用斜杠<br/>（同上） | 编辑者写的：`这里声母变成[/s]` <br/>用户看到的：`这里的声母变成 s` | 系统不会修改格式，只会修改字体                    |

## 不显示的内容：使用`/**/`

| 名称 | 结构范例            | 显示效果                                                | 说明                                     |
|----|-----------------|-----------------------------------------------------|----------------------------------------|
| 注释 | 使用`/*`和`*/`包裹内容 | 编辑者写的：<br/>`方言大詞典 /*标题页*/`<br/> <br/> 用户看到的：`方言大词典` | 1. 编辑的时候的笔记，不会显示的<br/>2. 可以用于结构标注和校对随笔 |

## 特殊格式的内容：使用`{}`

| 名称             | 结构范例              | 显示效果                                                                                                                | 说明               |
|----------------|-------------------|---------------------------------------------------------------------------------------------------------------------|------------------|
| 关键词            | 使用大括号`{}`         | 编辑者写的：`{你}：代詞，用于…… `<br/>  用户看到的：<br/>词条「你」下的附注：《南昌方言词典》 你 代词，用于……                                                  | 用于自动绑定           |
| 关键词<br/>（多个）   | 使用大括号`{}`         | 编辑者写的：`{医生}：又称{郎中}`<br/> <br/> 用户看到的：<br/> 词条「医生」下的附注：《南昌方言词典》 医生：又称郎中<br/>词条「郎中」下的附注：《南昌方言词典》 医生：又称郎中              | 直接可以多个绑定         |
| 关键词<br/>（隐藏绑定） | 关键词前面加横杠          | 编辑者写的：`{中秋}：……{-中秋节}`<br/>用户看到的：<br/> 词条「中秋」下的附注：《南昌方言词典》 中秋：……<br/>词条「中秋节」下的附注：《南昌方言词典》 中秋：……                      | 关键词不一定会出现在内容里    |
| 链接             | 用`l`标识（link）      | 编辑者写的：  `褡：见词条{l 褙褡}`<br/>  用户看到的：褡：见词条 [褙褡](标记语言规范：写法.md)                                                          |                  |
| 粗体             | 用`b`标识（bold）      | 编辑者写的： `这里用法{b 不同于}普通话` <br/> 用户看到的：这里用法**不同于**普通话                                                                  |                  |
| 原文注释           | 用`n`标识（note）      | 编辑者写的： `你慢慢子{n 慢慢儿的}来` <br/> 用户看到的：你慢慢子<small style="color: gray;">（原书注：慢慢儿的）                                       | 和注释的不同点在于这个是要显示的 |
| 同音字表           | 用`h`标识（homophone） | 编辑者写的： `{h [qyu1]蛆趋区驱蛐}`  <br/>  用户看到的：<br/>汉字「区」下的附注：《南昌方言词典》`qǜ` 蛆 趋 区 驱 蛐  <br/>汉字「驱」下的附注：《南昌方言词典》`qǜ` 蛆 趋 区 驱 蛐 |                  |
