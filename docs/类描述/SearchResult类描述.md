## 对于汉字

| 字段        | 信息                          | 说明          |
|-----------|-----------------------------|-------------|
| `title`   | 汉字                          |             |
| `explain` | 描述读音个数，是否特殊                 |             |
| `tag`     | 字符串"hanzi"                  | 前端识别hanzi标签 |
| `info`    | hanzi:对应汉字<br/>lang:刚刚查询的语言 | 这是反查的两个信息   |
