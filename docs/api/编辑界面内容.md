## 以下内容，只会在编辑界面显示，不会在用户的结果里显示，

### `/*注释*/`

一些不在原来书籍的内容，做补充说明和检索用，例如：

```
xxxxxx /*这个词条还需要修改，原文模糊不清*/
```

就可以没有额外担心的做标记了。

```
打平火   xxxxxxx /*分账吃饭，AA制*/
```

就可以用AA来查询了

### `++精确识别++`

一些高频字词，本身就会在词条中大量存在，这时候这个词条编辑的时候就要加上，避免被淹没,
```你們 [n3][meen0] 代詞，稱不止一個的對方或包括對方在內的若干人 ++你們++```
查询的时候搜索 ++你們++ ，就能准确找到这个词条

### `[拼音]`

输入国际音标很折磨人，所以直接使用方言音标处理
