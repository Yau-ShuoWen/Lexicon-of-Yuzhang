# 概述

方言模糊匹配字表，`xxx`为当地方言代号，详见[这篇文档](../../命名/方言代号命名.md)

# 结构

| 列名      | 内容         | 数据说明     |
|---------|------------|----------|
| id      | 🔍 自增主键    | 自动生成列    |
| char_id | 🔍 外键      | 对应方言主表id |
| sc      | 📖 汉字-简体中文 |          |
| tc      | 📖 汉字-繁体中文 |          |

# 数据限定

- 汉字和拼音组合在一起为一个存储单位。同汉字的不同读音被视作不同的条目。
- 所以：数据库使用的是`UNIQUE (hanzi, pinyin, zhuyin, tone)`作为唯一键，info也是唯一的，可以完全代表这个条目。
