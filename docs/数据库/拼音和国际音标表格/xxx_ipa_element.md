# 概述

方言国际音标音段表，`xxx`为当地方言代号，详见[这篇文档](../../命名/方言代号命名.md)

# 表结构

| 列名         | 内容    | 数据说明     | 内容说明                     |
|------------|-------|----------|--------------------------|
| `code`     | 编码    |          | 见拼音说明<br/>对于没有定义的部分使用`~` |
| `standard` | 声母/韵母 |          | 使用输入版本                   |
| 字典名        | 国际音标  |          | 用于观察，不用于查询，因为各异          |
| 字典名_y      | 严式音标  |          | 用于观察，不用于查询，因为各异          |
| `info`     | 信息列   | `JSON`对象 | 实际中时候使用这个列查询内容           |
