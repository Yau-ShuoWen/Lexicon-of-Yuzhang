# 音调

1. 如果有`a` `o` `e`，标注在最后面的一个`a` `o` `e`上，双写也是标在后一个
2. 没有，但有`i` `u` `ü`，标在最前面的`i` `u` `ü`上面，双写也是标在前一个
3. 如果都没有，只有三种情况：`m`标在`m`上、`n`标在`n`上、`ng`标在`n`上。
