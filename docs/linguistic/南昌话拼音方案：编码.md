# 号码

- ### 前两位：声母

| 编码   | 对应音节 |
|:-----|:-----|
| `00` | 零声母  |
| `01` | `b`  |
| `02` | `p`  |
| `03` | `m`  |
| `04` | `f`  |
| `05` | `d`  |
| `06` | `t`  |
| `07` | `l`  |
| `08` | `g`  |
| `09` | `k`  |
| `10` | `ng` |
| `11` | `h`  |
| `12` | `j`  |
| `13` | `q`  |
| `14` | `n`  |
| `15` | `x`  |
| `16` | `z`  |
| `17` | `c`  |
| `18` | `s`  |

- ### 第三位：韵尾

| 编码  | 对应音节      |
|:----|:----------|
| `0` | 开元音韵尾/无韵尾 |
| `1` | z c s的韵母  |
| `3` | i 尾       |
| `4` | u 尾       |
| `5` | n 尾       |
| `6` | ng 尾      |
| `7` | t 尾       |
| `8` | k 尾       |
| `9` | l 尾       |

- ### 第四位：介韵母

| 编码  | 对应音节    |
|:----|:--------|
| `0` | 开口呼     |
| `1` | 合口呼：`i` |
| `2` | 闭口呼：`u` |
| `3` | 撮口呼：`ü` |

- ### 第五位：中心元音

| 编码  | 对应音节      | 说明                                    |
|:----|:----------|---------------------------------------|
| `0` | 无主元音      | 诸如`iu` `ui`等<br/>前一个被认为是介韵母，后一个被认为是韵尾 |
| `1` | `a`       |                                       |
| `2` | `o`       |                                       |
| `3` | `e`       |                                       |
| `4` | `ee`      |                                       |
| `5` | `oe`      |                                       |
| `6` | `u`       |                                       |
| `7` | 自成音节的`m`  | `m`的编码就是00007                         |
| `8` | 自成音节的`n`  | `n`的编码就是00008                         |
| `9` | 自成音节的`ng` | `ng`的编码就是00009                        |

1. 如果有aoe，标注在最后面的一个aoe上，双写也是标在后一个
2. 没有，但有iuv，标在最前面的iuv上面，双写也是标在前一个
3. 特殊情况，`ng` `n`标在`n`上面，`m` 标在`m`上面

# 拼音可选参数

ü 的处理方式（参数yu）

| 代号 | 改成 | 说明    |
|----|----|-------|
| 0  | yu | 打字，原版 |
| 1  | ü  | 书写    |
| 2  | v  | （备用）  |
| 3  | ụ  | （备用）  |

ẹ 的处理方式（参数ee）

| 代号 | 改成 | 说明    |
|----|----|-------|
| 0  | ee | 打字，原版 |
| 1  | ẹ  | 书写    |
| 2  | ё  | （备用）  |

ọ 的处理方式（参数oe）

| 代号 | 改成 | 说明    |
|----|----|-------|
| 0  | oe | 打字，原版 |
| 1  | ọ | 书写    |
| 2  | ö  | （备用）  |
| 3  | o  | （备用）  |

i 的处理方式（参数ii）

| 代号 | 改成 | 说明    |
|----|----|-------|
| 0  | ii | 打字，原版 |
| 1  | i  | 书写    |
| 2  | 空  | （备用）  |
| 3  | ị  | （备用）  |

入声韵尾的处理方式（参数ptk）

| 代号 | 改成  | 说明       |
|----|-----|----------|
| 0  | t/k | 打字，书写，原版 |
| 1  | 省略  | （备用）     |
| 2  | h/h | （备用）     |
| 3  | q/q | （备用）     |
| 4  | t/h | （备用）     |

零声母的iu处理方式（参数yw）

| 代号 | 改成    | 说明          |
|----|-------|-------------|
| 0  | 不变    | 打字，书写，原版    |
| 1  | 符合普通话 | （备用）        |
| 2  | 直接加   | （备用，可能用语Yi） |

大写（参数capital）

| 代号 | 改成    | 说明 |
|----|-------|----|
| 0  | 全部小写  |    |
| 1  | 全部大写  |    |
| 2  | 首字母大写 |    |

标调方式

| 代号 | 改成         | 说明 |
|----|------------|----|
| 0  | 不显示        |    |
| 1  | 符号调标注在对应位置 | 书写 |
| 2  | 数字调加载后面    | 打字 |
| 3  | 符号调加载后面    |    |

i u优先级

默认顺序 `a > o > e > i > u > ü`

| 代号 | 改成     | 说明    |
|----|--------|-------|
| 0  | 没有特殊规则 |       |
| 1  | 并列标在后  | 例外是iu |
| 2  | 并列标在前  | 例外是ui |
